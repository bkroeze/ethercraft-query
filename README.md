# etheremonQ

A simple command-line query tool for Ethercraft inventory and sales.

## Installation

```
git clone https://github.com/bkroeze/ethercraftQ
npm install
#or
yarn install
```

## Usage

```
/path/to/ethercraftQ/ethercraftQ.js balance -a your-eth-account-address
```
